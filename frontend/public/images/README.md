# Image Assets

동그라미 분식 웹페이지에서 사용할 이미지 폴더입니다.

## Recommended Structure

- `store/`: 매장 외관, 내부, 간판 사진
- `menu/`: 떡볶이, 순대, 튀김 등 메뉴 사진
- `branding/`: 로고, 스티커, 포스터 같은 브랜딩 이미지

## Example Paths

- `/images/store/exterior.jpg`
- `/images/store/interior-01.jpg`
- `/images/menu/tteokbokki.jpg`
- `/images/menu/sundae.jpg`
- `/images/menu/fried-mix.jpg`
